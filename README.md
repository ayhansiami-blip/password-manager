# Password Manager V1.0

My first Python CLI project!

A simple and beginner-friendly Password Manager built with Python. This project helped me practice:

- Loops
- Conditionals
- Functions (coming soon!)
- File Handling
- Exception Handling
- Working with text files
- Git & GitHub

## Features

- Add a new password
- View all saved passwords
- Search passwords by website
- Search passwords by username
- Hide passwords when viewing them
- Store passwords locally in a text file

## Future Plans

- Master Password system
- Password Generator
- Password Encryption
- GUI Version
- Better Search System
- JSON support
- Improved UI/UX

## How to Run

```bash
python main.py
```

## Notes

- This project is for educational purposes.
- Passwords are currently stored locally in a text file and are NOT encrypted.
- Do not use it to store sensitive or real-world passwords.
